# Burger App - Node Express Handlebars

## Overview

Hamburger tracker, I.e., an app which allows you to add and devour virtual hamburgers utilizing MySQL, Node, Express, Handlebars and ORM, following MVC design principles.

## Technological Inventory

* [MySQL](https://www.mysql.com/)
* [Node](https://nodejs.org/en/)
* [Express](https://expressjs.com/)
* [Handlebars](https://www.npmjs.com/package/express-handlebars)
* [jQuery](https://jquery.com/)
* [Bootstrap](https://getbootstrap.com/)
